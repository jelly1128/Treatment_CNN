# プロジェクト設定

## コード編集のルール

- ファイルを編集する前に、**変更内容・変更理由・対象ファイル**を説明し、確認を取ること
- 複数ファイルにまたがる変更を行う場合は、先に全体の変更計画を示すこと
- 依頼された箇所以外は編集しないこと（コメント追加・リファクタ・スタイル修正なども含む）
- git commit は明示的に依頼されたときのみ行うこと

## プロジェクト情報

- Python 3.10 / PyTorch
- 仮想環境: `env/`
- 学習実行: `python train.py -c config/train_debug.yaml`
- テスト実行: `python test.py -c config/test_debug.yaml`
